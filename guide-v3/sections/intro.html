SDK آرایش مجازی v3

قدرتمندترین کتابخانه آرایش مجازی برای پروژه‌های وب با قابلیت اعمال آرایش در زمان واقعی.

ویژگی‌های کلیدی

  • Real-time: آرایش در زمان واقعی روی دوربین
  • پردازش تصویر: آپلود و آرایش روی عکس
  • Pattern Support: انواع pattern های آرایش
  • Responsive: سازگار با همه دستگاه‌ها
  • React Ready: پشتیبانی کامل از React

پیش‌نیازها

موارد ضروری:

  • Token: کلید احراز هویت از ARmo
  • Product UID: شناسه محصول آرایشی
  • HTTPS: برای دسترسی به دوربین
  • مرورگر مدرن: Chrome 60+, Firefox 55+, Safari 11+

نمونه سریع

HTML 
<link rel="stylesheet" href="https://cdn.makeup.armogroup.tech/sdk/v3/makeup.min.css">
<script src="https://cdn.makeup.armogroup.tech/sdk/v3/makeup.min.js"></script>
<div id="armo-makeup-view"></div>
JavaScript 
// تنظیم Token
window.Makeup.setToken('YOUR_TOKEN');

// راه‌اندازی SDK
const makeup = window.initMakeup({
    productUid: 'YOUR_PRODUCT_UID',
    mode: 'camera',
    onReady: () => console.log('آماده!')
});

// تغییر رنگ
makeup.changeMakeupColor('lips', '#FF69B4');

انواع آرایش پشتیبانی شده

  • lips: رژ لب
  • eyeshadow: سایه چشم
  • eyelashes: مژه
  • blush: رژگونه
  • foundation: کرم پودر
  • brows: ابرو
  • eyeliner: خط چشم
  • concealer: کانسیلر
  • lens: لنز رنگی
نکته مهم: برای دریافت Token و Product UID، باید در سایت dashboard.makeup.armogroup.tech ثبت‌نام کنید و محصول خود را ایجاد کنید.
