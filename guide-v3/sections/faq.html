سوالات متداول

پاسخ به رایج‌ترین سوالات درباره SDK آرایش مجازی v3

نصب و راه‌اندازی

❓ چطور Token و Product UID دریافت کنم؟

پاسخ: باید در سایت ARmo ثبت‌نام کنید، محصول بسازید و از پنل مدیریت Token و Product UID را کپی کنید.

❓ SDK روی HTTP کار می‌کند؟

پاسخ: خیر، به دلیل امنیت دوربین، فقط روی HTTPS کار می‌کند. localhost مستثنی است.

❓ چه مرورگرهایی پشتیبانی می‌شوند؟

پاسخ: Chrome 60+، Firefox 55+، Safari 11+، Edge 79+. Internet Explorer پشتیبانی نمی‌شود.

عملکرد و مشکلات

❓ دوربین کار نمی‌کند، چه کنم؟

پاسخ:

  • مطمئن شوید روی HTTPS هستید
  • اجازه دسترسی به دوربین را بدهید
  • دوربین توسط برنامه دیگری استفاده نشود
  • Console مرورگر را برای خطاها چک کنید

❓ SDK کند کار می‌کند، چرا؟

پاسخ:

  • اتصال اینترنت را بررسی کنید
  • از مرورگر به‌روز استفاده کنید
  • Tab های اضافی را ببندید
  • سخت‌افزار ضعیف ممکن است تأثیر بگذارد

❓ چهره تشخیص داده نمی‌شود؟

پاسخ:

  • نور کافی داشته باشید
  • مستقیم به دوربین نگاه کنید
  • فاصله مناسب با دوربین داشته باشید
  • موهایتان صورت را نپوشاند

توسعه و کدنویسی

❓ چطور در React استفاده کنم؟

پاسخ: از Hook که در بخش React توضیح داده شده استفاده کنید و cleanup را فراموش نکنید.

React Hook 
const { makeup, isReady } = useMakeup(token, productUid);

useEffect(() => {
    return () => makeup?.cleanup();
}, [makeup]);

❓ چطور خطاها را مدیریت کنم؟

پاسخ: از callback های onError استفاده کنید:

Error Handling 
window.initMakeup({
    productUid: 'YOUR_UID',
    onError: (error) => {
        if (error.name === 'NotAllowedError') {
            alert('لطفاً اجازه دسترسی به دوربین را بدهید');
        } else if (error.name === 'NotFoundError') {
            alert('دوربین یافت نشد');
        } else {
            console.error('خطا:', error);
        }
    }
});

❓ چطور محصول را تغییر دهم؟

پاسخ: از متد loadProduct استفاده کنید:

Change Product 
// بارگذاری محصول جدید
const newProduct = await makeup.loadProduct('NEW_PRODUCT_UID');
if (newProduct) {
    console.log('محصول تغییر یافت:', newProduct.name);
}

محدودیت‌ها و قیمت‌گذاری

❓ نسخه رایگان چه محدودیت‌هایی دارد؟

پاسخ:

  • لوگوی "Powered by ARmo" نمایش داده می‌شود
  • تعداد درخواست محدود است
  • برخی ویژگی‌های پیشرفته در دسترس نیست

❓ چطور نسخه Premium بگیرم؟

پاسخ: از طریق سایت ARmo می‌توانید نسخه Premium خریداری کنید که شامل:

  • حذف لوگو
  • درخواست‌های نامحدود
  • تمام ویژگی‌ها
  • پشتیبانی اولویت‌دار

مشکلات رایج

❓ SDK لود نمی‌شود

راه‌حل:

Debug Loading 
// بررسی لود شدن SDK
if (typeof window.Makeup === 'undefined') {
    console.error('SDK لود نشده - script tag را چک کنید');
} else {
    console.log('SDK لود شده');
    window.Makeup.setToken('YOUR_TOKEN');
}

❓ Token نامعتبر است

راه‌حل:

  • Token را از پنل ARmo مجدداً کپی کنید
  • فاصله اضافی در Token نباشد
  • تاریخ انقضا را بررسی کنید
  • Token را قبل از initMakeup set کنید

❓ رنگ‌ها اعمال نمی‌شوند

راه‌حل:

Check Status 
// بررسی آماده بودن SDK
if (makeup.getStatus() === 'ready') {
    makeup.changeMakeupColor('lips', '#FF0000');
} else {
    console.log('SDK هنوز آماده نیست');
}

پشتیبانی

🆘 نیاز به کمک دارید؟

  • مستندات: این راهنما را کاملاً مطالعه کنید
  • Console: خطاهای مرورگر را بررسی کنید
  • تست: روی localhost ابتدا تست کنید
  • تماس: از طریق سایت armogroup.tech با پشتیبانی تماس بگیرید

⚠️ نکات مهم:

  • هیچ‌وقت Token را در کد عمومی قرار ندهید
  • از متغیرهای محیطی استفاده کنید
  • SDK را فقط از CDN رسمی لود کنید
  • cleanup را فراموش نکنید
