سوالات عمومی

آیا می‌توانم SDK را به صورت آفلاین استفاده کنم؟

خیر، SDK نیاز به اتصال اینترنت دارد زیرا باید:

  • توکن به صورت آنلاین اعتبارسنجی شود
  • اطلاعات محصول از سرور بارگیری شود
  • منابع مورد نیاز از CDN بارگیری شوند

چه مرورگرهایی پشتیبانی می‌شوند؟

مرورگرهای مدرن با نسخه‌های زیر پشتیبانی می‌شوند:

  • Chrome نسخه 88 و بالاتر
  • Firefox نسخه 85 و بالاتر
  • Safari نسخه 14 و بالاتر
  • Edge نسخه 88 و بالاتر

تفاوت نسخه 3 با نسخه‌های قبلی چیست؟

تغییرات مهم در نسخه 3:

  • ساده‌سازی API و حذف پیچیدگی‌های غیرضروری
  • معرفی مفهوم Product UID برای مدیریت محصولات
  • حذف نیاز به تعریف دستی رنگ‌ها و pattern ها
  • بهبود عملکرد و سرعت
  • پشتیبانی بهتر از حالت تصویر

سوالات فنی

چرا دوربین کار نمی‌کند؟

موارد زیر را بررسی کنید:

  • مجوز دسترسی به دوربین در مرورگر
  • اتصال HTTPS در محیط وب
  • عدم استفاده همزمان دوربین توسط برنامه‌های دیگر
// بررسی دسترسی به دوربین
if (!navigator.mediaDevices || !navigator.mediaDevices.getUserMedia) {
    console.error("دوربین در این مرورگر پشتیبانی نمی‌شود");
}

چگونه می‌توانم کیفیت تشخیص چهره را بهبود دهم؟

  1. از نور کافی و مناسب استفاده کنید
  2. مستقیم به دوربین نگاه کنید
  3. فاصله مناسب (30-60 سانتیمتر) را حفظ کنید
  4. از پس‌زمینه ساده استفاده کنید

چگونه وضعیت SDK را مانیتور کنم؟

// روش 1: استفاده از getStatus
const status = makeup.getStatus();
console.log("وضعیت SDK:", status);

// روش 2: استفاده از callbacks
const makeup = window.initMakeup({
    productUid: "YOUR_PRODUCT_UID",
    onReady: () => console.log("SDK آماده است"),
    onError: (error) => console.error("خطا:", error)
});

چگونه بین حالت دوربین و تصویر جابجا شوم؟

// تغییر به حالت تصویر
await makeup.switchMode('image');

// تغییر به حالت دوربین
await makeup.switchMode('camera');

// بررسی حالت فعلی
const info = makeup.getTechnicalInfo();
console.log("حالت فعلی:", info.currentMode);

سوالات مربوط به محصولات

Product UID چیست و چگونه آن را دریافت کنم؟

Product UID شناسه یکتای محصول آرایشی شماست که شامل:

  • تعریف انواع آرایش مجاز
  • رنگ‌ها و pattern های هر آرایش
  • تنظیمات خاص محصول

برای دریافت Product UID:

  1. وارد پنل مدیریت ARmo شوید
  2. محصول جدید ایجاد کنید
  3. Product UID را از صفحه محصول کپی کنید

چگونه محصول را در زمان اجرا تغییر دهم؟

// بارگذاری محصول جدید
const newProduct = await makeup.loadProduct('NEW_PRODUCT_UID');

if (newProduct && newProduct.success) {
    console.log('محصول جدید لود شد:', newProduct.name);
    console.log('ویژگی‌های جدید:', newProduct.features);
} else {
    console.error('خطا در بارگذاری محصول جدید');
}

چگونه بدانم چه ویژگی‌هایی در محصول فعلی فعال است؟

// دریافت ویژگی‌های فعال
const features = makeup.getAvailableFeatures();
console.log('ویژگی‌های مجاز:', features);

// بررسی ویژگی خاص
if (makeup.isFeatureEnabled('eyeshadow')) {
    // سایه چشم فعال است
    makeup.changeMakeupColor('eyeshadow', '#8A2BE2');
}

سوالات پیاده‌سازی

چگونه می‌توانم از رنگ‌های سفارشی استفاده کنم؟

در نسخه 3، رنگ‌ها از طریق محصول تعریف می‌شوند، اما می‌توانید:

// تغییر مستقیم رنگ
makeup.changeMakeupColor("lips", "#FF0000");

// استفاده از رنگ‌های از پیش تعریف شده محصول
const info = makeup.getTechnicalInfo();
console.log("رنگ‌های موجود:", info.availableColors);

چگونه در React از SDK استفاده کنم؟

بهترین روش استفاده از Hook سفارشی است:

const { makeup, isReady, error } = useMakeup(token, productUid);

// cleanup را فراموش نکنید
useEffect(() => {
    return () => makeup?.cleanup();
}, [makeup]);

چگونه لودینگ سفارشی تعریف کنم؟

با استفاده از callback های onReady:

const makeup = window.initMakeup({
    productUid: "YOUR_PRODUCT_UID",
    onReady: () => {
        // مخفی کردن لودینگ سفارشی
        document.getElementById("my-loading").style.display = "none";
    }
});

عیب‌یابی

پیام "توکن نامعتبر است" دریافت می‌کنم

  1. از معتبر بودن توکن اطمینان حاصل کنید
  2. تاریخ انقضای توکن را بررسی کنید
  3. توکن را قبل از initMakeup تنظیم کنید: 
    window.Makeup.setToken("YOUR_TOKEN");
const makeup = window.initMakeup(...);

پیام "Product UID نامعتبر است" دریافت می‌کنم

  • Product UID را از پنل ARmo مجدداً کپی کنید
  • مطمئن شوید محصول منتشر شده باشد
  • بررسی کنید محصول حذف نشده باشد

در موبایل تصویر وارونه نمایش داده می‌شود

این مشکل با افزودن CSS زیر برطرف می‌شود:

.armo-sdk-input-video {
    transform: scaleX(-1);
}

SDK لود نمی‌شود یا کار نمی‌کند

// بررسی لود شدن SDK
if (typeof window.Makeup === 'undefined') {
    console.error('SDK لود نشده - script tag را چک کنید');
} else {
    console.log('SDK لود شده');
    window.Makeup.setToken('YOUR_TOKEN');
}

// بررسی وضعیت
const status = makeup?.getStatus();
console.log('وضعیت SDK:', status);
