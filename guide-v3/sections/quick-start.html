شروع سریع

در کمتر از 5 دقیقه SDK را راه‌اندازی کنید.

مرحله 1: اضافه کردن SDK

فایل‌های CSS و JS

این کدها را به HTML خود اضافه کنید:

HTML 
<!-- CSS -->
<link rel="stylesheet" href="https://cdn.makeup.armogroup.tech/sdk/v3/makeup.min.css">

<!-- JavaScript -->
<script src="https://cdn.makeup.armogroup.tech/sdk/v3/makeup.min.js"></script>

مرحله 2: HTML ساختار

المنت مورد نیاز

HTML 
<div id="armo-makeup-view"></div>

مرحله 3: راه‌اندازی

تنظیم Token و ProductUID

JavaScript 
// تنظیم Token
window.Makeup.setToken('YOUR_TOKEN_HERE');

// راه‌اندازی SDK
const makeup = window.initMakeup({
    productUid: 'YOUR_PRODUCT_UID',
    onReady: () => {
        console.log('SDK آماده است!');
    },
    onError: (error) => {
        console.error('خطا:', error);
    }
});

مرحله 4: استفاده از API

کنترل آرایش

JavaScript 
// تغییر رنگ آرایش
makeup.changeMakeupColor('lips', '#FF6B9D');

// تغییر pattern آرایش
makeup.changeMakeupPattern('lips', 'matte');

// تغییر حالت به تصویر
await makeup.switchMode('image');

// بارگذاری محصول جدید
await makeup.loadProduct('NEW_PRODUCT_UID');

مثال کامل

index.html 
<!DOCTYPE html>
<html lang="fa">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>آرایش مجازی</title>
    
    <link rel="stylesheet" href="https://cdn.makeup.armogroup.tech/sdk/v3/makeup.min.css">
    <style>
        body { margin: 0; padding: 20px; font-family: Arial; }
        .container { max-width: 800px; margin: 0 auto; }
        .controls { padding: 20px; text-align: center; }
        .btn { background: #007bff; color: white; border: none; 
               padding: 10px 20px; margin: 5px; border-radius: 5px; cursor: pointer; }
    </style>
</head>
<body>
    <div class="container">
        <h1>آرایش مجازی</h1>
        
        <div id="armo-makeup-view"></div>
        
        <div class="controls">
            <button class="btn" onclick="changeColor('#FF6B9D')">صورتی</button>
            <button class="btn" onclick="changeColor('#DC143C')">قرمز</button>
            <button class="btn" onclick="changePattern('matte')">مات</button>
            <button class="btn" onclick="changePattern('glossy')">براق</button>
        </div>
    </div>

    <script src="https://cdn.makeup.armogroup.tech/sdk/v3/makeup.min.js"></script>
    <script>
        let makeupInstance;
        
        window.addEventListener('DOMContentLoaded', () => {
            // تنظیم Token
            window.Makeup.setToken('YOUR_TOKEN_HERE');
            
            // راه‌اندازی
            makeupInstance = window.initMakeup({
                productUid: 'YOUR_PRODUCT_UID',
                onReady: () => console.log('آماده!'),
                onError: (error) => alert('خطا: ' + error.message)
            });
        });
        
        function changeColor(color) {
            makeupInstance?.changeMakeupColor('lips', color);
        }
        
        function changePattern(pattern) {
            makeupInstance?.changeMakeupPattern('lips', pattern);
        }
    </script>
</body>
</html>

نکات مهم:

  • Token و Product UID را از پنل ARmo دریافت کنید
  • حتماً روی HTTPS تست کنید
  • اجازه دسترسی به دوربین را بدهید
