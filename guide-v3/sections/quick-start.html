شروع سریع

1. دریافت توکن و Product UID

برای استفاده از SDK نسخه 3، نیاز به دو مورد دارید:

  • Token: کلید احراز هویت شما
  • Product UID: شناسه محصول آرایشی که می‌خواهید استفاده کنید

نکته: هر دو مورد از طریق پنل مدیریت ARmo قابل دریافت هستند. ابتدا محصول خود را ایجاد کنید سپس Product UID آن را کپی کنید.

2. افزودن کتابخانه

فایل‌های مورد نیاز را به پروژه خود اضافه کنید:

<link rel="stylesheet" href="https://cdn.makeup.armogroup.tech/sdk/v3/makeup.min.css">
<script src="https://cdn.makeup.armogroup.tech/sdk/v3/makeup.min.js"></script>

3. راه‌اندازی اولیه

3.1. افزودن المان پایه

<div id="armo-makeup-view"></div>

3.2. راه‌اندازی SDK

document.addEventListener("DOMContentLoaded", () => {
    // تنظیم توکن
    window.Makeup.setToken("YOUR_TOKEN_HERE");

    // راه‌اندازی SDK
    const makeup = window.initMakeup({
        productUid: "YOUR_PRODUCT_UID",
        onReady: () => {
            console.log("SDK آماده استفاده است");
        },
        onError: (error) => {
            console.error("خطا در SDK:", error);
        }
    });

    // ذخیره reference برای استفاده بعدی
    window.makeupInstance = makeup;
});

گزینه‌های پیکربندی

گزینه نوع پیش‌فرض توضیحات
productUid String - شناسه محصول (اجباری)
mode String "camera" حالت کار: camera یا image
showColorPicker Boolean true نمایش/عدم نمایش انتخابگر رنگ داخلی
transparency Number 0.8 شفافیت کلی آرایش (0 تا 1)
onReady Function () => {} تابع callback برای زمان آماده شدن SDK
onError Function () => {} تابع callback برای زمان بروز خطا

بررسی وضعیت SDK

می‌توانید وضعیت فعلی SDK را با استفاده از متد getStatus() بررسی کنید: 

const status = makeup.getStatus();
// status می‌تواند یکی از این مقادیر باشد:
// - 'initializing': در حال راه‌اندازی اولیه
// - 'loading': در حال بارگذاری منابع
// - 'ready': آماده استفاده
// - 'error': خطا در راه‌اندازی
// - 'paused': متوقف شده
// - 'cleanup': در حال پاکسازی

4. استفاده اولیه

پس از راه‌اندازی موفق SDK، می‌توانید شروع به استفاده از قابلیت‌های آن کنید:

4.1. تغییر رنگ آرایش

// تغییر رنگ رژ لب
makeup.changeMakeupColor('lips', '#FF69B4');

// تغییر رنگ سایه چشم
makeup.changeMakeupColor('eyeshadow', '#8A2BE2');

// تغییر رنگ رژگونه
makeup.changeMakeupColor('blush', '#FF6B6B');

4.2. تغییر pattern آرایش

// تغییر pattern رژ لب به matte
makeup.changeMakeupPattern('lips', 'matte');

// تغییر pattern سایه چشم به shimmer
makeup.changeMakeupPattern('eyeshadow', 'shimmer');

4.3. تغییر حالت به تصویر

// تغییر به حالت تصویر
await makeup.switchMode('image');

// تغییر مجدد به حالت دوربین
await makeup.switchMode('camera');
