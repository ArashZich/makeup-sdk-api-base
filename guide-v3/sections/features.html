حالت‌های کار

SDK از دو حالت کار پشتیبانی می‌کند:

حالت توضیحات کاربرد
camera آرایش زنده روی دوربین آرایش در زمان واقعی
image آپلود و آرایش تصاویر ویرایش تصاویر موجود

نکته: می‌توانید در طول استفاده، بین این دو حالت جابجا شوید.

دوربین

SDK از تنظیمات استاندارد زیر برای دوربین استفاده می‌کند:

پارامتر مقدار
رزولوشن 1280×720 پیکسل
جهت دوربین دوربین جلو

نکته: در صورتی که دستگاه کاربر از این رزولوشن پشتیبانی نکند، به صورت خودکار از بهترین رزولوشن قابل دسترس استفاده خواهد شد.

تشخیص نور محیط

SDK به صورت خودکار شرایط نوری محیط را بررسی می‌کند:

  • آستانه روشنایی: 80 از 255
  • نمایش خودکار هشدار در شرایط کم‌نور
  • تنظیم خودکار پارامترهای تصویر

Callbacks

SDK دو callback اصلی را ارائه می‌دهد:

const makeup = window.initMakeup({
    productUid: 'YOUR_PRODUCT_UID',
    onReady: () => {
        console.log("SDK آماده استفاده است");
        // مخفی کردن loading
        document.getElementById("loading").style.display = "none";
    },
    onError: (error) => {
        console.error("خطا در SDK:", error);
        // نمایش پیغام خطا
        showError(error.message);
    }
});

مدیریت محصولات

یکی از ویژگی‌های کلیدی نسخه 3، قابلیت مدیریت محصولات مختلف است:

تغییر محصول در زمان اجرا

می‌توانید بدون راه‌اندازی مجدد SDK، محصول فعال را تغییر دهید.

// بارگذاری محصول جدید
const newProduct = await makeup.loadProduct('NEW_PRODUCT_UID');
if (newProduct) {
    console.log('محصول جدید لود شد:', newProduct.name);
}

اطلاعات محصول

دسترسی به اطلاعات تفصیلی محصول فعال:

const info = makeup.getTechnicalInfo();
console.log('محصول فعال:', info.currentProduct);
console.log('ویژگی‌های فعال:', info.enabledFeatures);
